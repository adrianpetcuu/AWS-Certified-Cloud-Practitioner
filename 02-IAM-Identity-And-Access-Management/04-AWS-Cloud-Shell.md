# AWS CloudShell (Short Overview)

## What is AWS CloudShell?

AWS CloudShell is a browser-based terminal that allows you to run AWS
CLI commands directly from the AWS Management Console.

You do not need to install or configure anything locally.

------------------------------------------------------------------------

## Key Features

-   Pre-installed AWS CLI
-   Pre-configured with your AWS credentials
-   Secure, browser-based access
-   Temporary storage (persists between sessions in the same region)

------------------------------------------------------------------------

## When to Use CloudShell

-   Quick testing of AWS CLI commands
-   Learning and practice
-   Troubleshooting
-   Running scripts without local setup

------------------------------------------------------------------------

## Benefits

-   No local installation required
-   Automatically authenticated with your IAM user
-   Secure and isolated environment

------------------------------------------------------------------------

## Summary

-   CloudShell = AWS CLI in the browser
-   No access keys needed locally
-   Ideal for quick commands and learning
